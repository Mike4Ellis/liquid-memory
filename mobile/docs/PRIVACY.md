# Privacy Policy - Liquid Memory

Last updated: February 7, 2026

## Overview

Liquid Memory is a local-first AI creative studio. Your privacy is our priority.

## Data Collection

### What We Collect
- **Account**: Anonymous user ID (no email required)
- **Content**: Images and prompts you create (stored locally by default)
- **Sync Data**: Only if you enable cloud sync (encrypted)

### What We Don't Collect
- Personal identifiers (name, email, phone)
- Location data
- Analytics or tracking data
- Third-party sharing

## Data Storage

### Local-First
All your creative content stays on your device by default.

### Optional Cloud Sync
- End-to-end encrypted (AES-GCM)
- Zero-knowledge architecture
- You control encryption keys

## Permissions

- **Camera**: To capture images for analysis
- **Photos**: To save and access your creative library
- **Notifications**: For daily inspiration reminders (optional)
